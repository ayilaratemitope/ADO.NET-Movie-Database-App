# ADO.NET Movie Database Application

## Overview
This is a C# console application that uses ADO.NET to interact with a SQL Server LocalDB database. 
The application allows users to manage movie records through a menu-driven interface.

## Features
- Add Movie
- List All Movies
- Find Movie by Title
- Find Movie by Id
- Update Movie by Id
- Delete Movie by Id

## Technologies Used
- C#
- ADO.NET
- SQL Server LocalDB
- Visual Studio

## Key Concepts
- SqlConnection
- SqlCommand
- SqlDataReader
- Parameterised Queries (SQL Injection prevention)

## How to Run
1. Open the solution file in Visual Studio
2. Ensure LocalDB is installed
3. Run the application
4. Use the menu options
